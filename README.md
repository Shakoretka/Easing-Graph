# Easing Graph

Easing functions library for Unity node based tools

![Easinf Preview](https://github.com/Shakoretka/shakoretka.github.io/blob/main/Drive/EasingPreview.webp?raw=true)

## Features
- 30 easing functions
- Custom HLSL + ShaderGraph/VFXGraph nodes
- VFXGraph/ShaderGraph compatible

## Installation

### Option 1 — Unity Package Manager (recommended)

Open **Window → Package Manager**  
Click **+ → Add package from git URL**

Paste:
https://github.com/Shakoretka/Easing-Graph.git

---

### Option 2 — UnityPackage

Download the latest .unitypackage from Releases:
https://github.com/Shakoretka/Easing-Graph/releases

Import it via:
Assets → Import Package → Custom Package
